PERTEMUAN 00 · BAGIAN 05

CSS Concepts

Variables, Flexbox, Grid & Responsive Design

Analogi Dunia Nyata

🎯 Memahami CSS dengan Analogi

CSS Variables = Cat Dinding 🎨

Kamu pilih warna sekali di toko cat (:root), lalu pakai warna itu di semua ruangan (var(--warna)). Kalau bosan, cukup ganti di satu tempat — semua ruangan ikut berubah!

Flexbox = Rak Buku Horizontal 📚

Buku-buku otomatis berderet rapi ke samping. Kalau raknya penuh, buku bisa turun ke baris berikutnya. Kamu cuma atur arah deretnya.

CSS Grid = Rak dengan Baris DAN Kolom 🗄️

Bayangkan rak pajangan. Kamu bisa atur berapa baris, berapa kolom, dan setiap barang punya “slot” sendiri. Lebih terstruktur dari Flexbox.

Responsive Design = Furniture Lipat 🪑

Seperti meja lipat yang menyesuaikan ukuran ruangan. Di ruangan besar, meja terbuka penuh. Di ruangan kecil, meja dilipat jadi lebih compact.

Bagian 5A

📖 CSS Variables

CSS Variables (Custom Properties) memungkinkan kamu menyimpan nilai yang bisa dipakai ulang di mana saja dalam stylesheet. Definisikan sekali, pakai berkali-kali!

Apa Itu CSS Variables?

CSS Variables adalah properti khusus yang dimulai dengan --. Biasanya didefinisikan di :root agar bisa diakses secara global.

Kenapa Pakai Variables?

  • Ubah satu nilai → berubah di mana-mana
  • Konsistensi warna, spacing, dan ukuran
  • Gampang bikin dark mode atau tema alternatif

Cara Menggunakan

styles.css — Definisi Variables
:root {
  /* Warna utama */
  --brand: #0891b2;
  --brand-light: rgba(8, 145, 178, 0.08);
  /* Spacing */
  --gap-sm: 8px;
  --gap-md: 16px;
  --gap-lg: 32px;
  /* Shadow */
  --shadow: 0 2px 12px rgba(0,0,0,.08);
}
styles.css — Penggunaan Variables
.card {
  background: var(--brand-light);
  border: 1px solid var(--brand);
  padding: var(--gap-md);
  box-shadow: var(--shadow);
}
.button {
  background: var(--brand);
  color: white;
  padding: var(--gap-sm) var(--gap-md);
}

Diagram: Alur CSS Variables

💡 Tips Pakai naming convention yang jelas, misal --color-brand, --spacing-md. Ini memudahkan kamu dan tim kamu nanti!
Bagian 5B

⚙️ Flexbox

Flexbox adalah metode layout 1-dimensi — kamu mengatur elemen dalam satu arah: baris (row) ATAU kolom (column). Cocok untuk navigasi, card row, dan centering.

Properti Kunci

  • display: flex — aktifkan flexbox di container
  • gap — jarak antar item
  • justify-content — atur posisi horizontal (center, space-between, dll.)
  • align-items — atur posisi vertikal (center, stretch, dll.)
  • flex-wrap — izinkan item pindah baris kalau penuh

Animasi: Item Berderet

Contoh: Layout Navbar

navbar.css
.navbar {
  display: flex;
  justify-content: space-between;
  align-items: center;
  padding: 12px 24px;
  gap: 16px;
}
.navbar-links {
  display: flex;
  gap: 20px;
  list-style: none;
}
card-row.css
.card-row {
  display: flex;
  gap: 16px;
  flex-wrap: wrap;
}
.card-row .card {
  flex: 1 1 250px; /* min 250px, bisa stretch */
}
Bagian 5C

🔲 CSS Grid

CSS Grid adalah layout 2-dimensi — kamu bisa mengatur baris DAN kolom sekaligus. Cocok untuk dashboard, gallery, dan layout kompleks.

Properti Kunci

  • display: grid — aktifkan grid di container
  • grid-template-columns — tentukan jumlah dan ukuran kolom
  • gap — jarak antar item
  • repeat() — shorthand untuk kolom berulang
  • 1fr — fraction unit, membagi ruang secara proporsional
  • minmax() — ukuran minimum dan maksimum

Animasi: Item Masuk ke Grid

Contoh: Dashboard Layout

dashboard.css
.dashboard {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 20px;
  padding: 20px;
}
/* Responsive: 1 kolom di mobile */
@media (max-width: 600px) {
  .dashboard {
    grid-template-columns: 1fr;
  }
}
responsive-grid.css
/* Grid yang otomatis responsive */
.auto-grid {
  display: grid;
  grid-template-columns: repeat(
    auto-fill, minmax(250px, 1fr)
  );
  gap: 16px;
}
💡 Pro Tip Gunakan auto-fill dengan minmax() untuk grid yang responsive tanpa perlu media query! Ini trik favorit developer profesional.
Bagian 5D

📱 Responsive Design

Responsive design artinya desain yang menyesuaikan diri dengan ukuran layar pengguna — dari monitor besar sampai HP kecil.

Pendekatan Mobile-First

Mulai desain dari layar kecil dulu (HP), lalu tambahkan styling untuk layar lebih besar. Ini lebih efisien karena HP biasanya punya resource terbatas.

Breakpoint Umum

breakpoints.css
/* Base: mobile (default) */
.container {
  padding: 16px;
}
/* Tablet: ≥ 600px */
@media (min-width: 600px) {
  .container { padding: 24px; }
}
/* Laptop: ≥ 900px */
@media (min-width: 900px) {
  .container { max-width: 860px; }
}
/* Desktop: ≥ 1200px */
@media (min-width: 1200px) {
  .container { max-width: 1100px; }
}

Contoh: Grid Responsif

responsive-cards.css
.card-grid {
  display: grid;
  gap: 16px;
  grid-template-columns: 1fr; /* Mobile: 1 kolom */
}
@media (min-width: 600px) {
  .card-grid {
    grid-template-columns: repeat(2, 1fr); /* Tablet: 2 kolom */
  }
}
@media (min-width: 900px) {
  .card-grid {
    grid-template-columns: repeat(3, 1fr); /* Desktop: 3 kolom */
  }
}

Animasi: Layar Berubah Ukuran

⚡ Penting Selalu tambahkan <meta name="viewport" content="width=device-width, initial-scale=1.0"> di <head>. Tanpa ini, responsive design tidak akan bekerja di HP!
Perbandingan

⚖️ Flexbox vs Grid

Keduanya berguna! Tapi kapan pakai yang mana? Cek tabel perbandingan berikut.

Aspek Flexbox CSS Grid
Dimensi 1D (baris ATAU kolom) 2D (baris DAN kolom)
Cocok untuk Navbar, card row, centering Dashboard, gallery, layout utama
Kontrol Konten menentukan ukuran Layout menentukan ukuran
Wrap flex-wrap Otomatis dengan auto-fill
Alignment justify-content, align-items place-items, place-content
Kapan pakai? Elemen berjajar satu arah Layout kompleks dua arah
💡 Tips Praktis Kamu bisa kombinasi keduanya! Gunakan Grid untuk layout halaman utama, lalu Flexbox untuk komponen kecil di dalamnya (misal navbar dalam grid area).
Hindari Ini

⚠️ Kesalahan Umum

Berikut 4 kesalahan yang sering dilakukan pemula. Jangan sampai kamu ikut-ikutan ya!

1. Menggunakan px untuk Semua Ukuran

font-size: 16px;
width: 960px;
padding: 20px;
Semua pakai px — tidak fleksibel di layar berbeda.
font-size: 1rem;
width: 90%; atau max-width: 60rem;
padding: 1.25rem;
Gunakan rem, %, dan vw untuk ukuran yang adaptif.

2. Menggunakan Margin Bukan Gap

.card { margin-right: 16px; }
.card:last-child { margin-right: 0; }
Perlu reset margin di elemen terakhir. Ribet!
.card-container { display: flex; gap: 16px; }
Gap otomatis hanya memberi jarak antar item. Simpel!

3. Lupa Testing di Mobile

grid-template-columns: repeat(4, 1fr);
4 kolom di HP? Konten bakal terlalu kecil dan susah dibaca.
grid-template-columns: repeat(auto-fill, minmax(250px, 1fr));
Otomatis menyesuaikan kolom berdasarkan lebar layar.

4. Layout Terlalu Kompleks

div > div > div > div > div { ... }
Nesting terlalu dalam. Susah di-debug dan di-maintain.
.layout { display: grid; }
.layout-item { ... }
Keep it simple! Struktur datar lebih gampang dikelola.
Praktik

✏️ Latihan

Coba kerjakan latihan berikut untuk mengasah skill CSS kamu. Tidak perlu pakai framework, cukup HTML dan CSS biasa.

Latihan 1

Buat CSS Variable System

Buat file variables.css yang mendefinisikan variable untuk website personal kamu:

  • Minimal 3 warna (brand, background, text)
  • 3 ukuran spacing (sm, md, lg)
  • 2 shadow (ringan dan kuat)
  • 1 border-radius

Lalu buat halaman sederhana yang menggunakan semua variable tersebut.

Latihan 2

Buat Navbar dengan Flexbox

Buat navigasi bar yang memiliki:

  • Logo di kiri, link navigasi di kanan
  • Gunakan display: flex dan justify-content: space-between
  • Link navigasi juga pakai flexbox dengan gap
  • Tambahkan hover effect pada setiap link
Latihan 3

Responsive Card Grid

Buat grid kartu profil yang:

  • 3 kolom di desktop (≥ 900px)
  • 2 kolom di tablet (≥ 600px)
  • 1 kolom di mobile (default)
  • Setiap kartu punya gambar, nama, dan deskripsi singkat
  • Gunakan gap dan CSS Variables untuk konsistensi
Rangkuman

📋 Ringkasan

Berikut poin-poin penting yang sudah kamu pelajari hari ini:

  • ✅ CSS Variables — Definisikan di :root, pakai dengan var(). Ubah satu tempat, berubah di mana-mana.
  • ✅ Flexbox — Layout 1-dimensi. Gunakan display: flex, atur dengan gap, justify-content, dan align-items.
  • ✅ CSS Grid — Layout 2-dimensi. Gunakan repeat() dan 1fr untuk kolom fleksibel.
  • ✅ Responsive Design — Pakai pendekatan mobile-first. Gunakan @media untuk breakpoint 600px, 900px, dan 1200px.
  • ✅ Gap bukan Margin — Selalu gunakan gap untuk jarak antar elemen di flex dan grid container.
  • ✅ rem bukan px — Gunakan unit relatif (rem, %, vw) untuk ukuran yang adaptif.
  • ✅ auto-fill + minmax — Trik grid responsif tanpa media query.
