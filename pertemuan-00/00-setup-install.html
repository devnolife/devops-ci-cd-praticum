Langkah 00 — WAJIB

Setup & Instalasi Next.js

Langkah pertama sebelum memulai semua materi. Pastikan selesai sebelum lanjut!

📋 Apa yang Kamu Butuhkan

Sebelum mulai, pastikan kamu sudah punya hal-hal berikut:

  • ✅ Komputer (Windows / macOS / Linux)
  • ✅ Koneksi internet (untuk download)
  • ✅ Text editor (VS Code — recommended)
  • ✅ Terminal / Command Prompt
💡 Tips: Kalau belum punya VS Code, download gratis di code.visualstudio.com

🔧 Langkah 1 — Install Node.js

Node.js adalah runtime JavaScript yang dibutuhkan untuk menjalankan Next.js. Download dari nodejs.org — pilih versi LTS (Long Term Support).

Setelah install, buka terminal dan cek versinya:

⚠️ Perhatian: Harus versi 18 atau lebih baru. Kalau versi lama, download ulang dari nodejs.org

🚀 Langkah 2 — Buat Proyek Next.js

Buka terminal, lalu jalankan perintah berikut untuk membuat proyek baru:

Terminal — buat proyek baru
npx create-next-app@latest nama-proyek-kamu

Akan muncul pertanyaan interaktif. Jawab seperti ini untuk menggunakan JavaScript (bukan TypeScript):

🚫 PENTING: Pilih NO untuk TypeScript! Kalau salah, hapus folder dan ulangi npx create-next-app@latest dengan pilihan yang benar.

📂 Langkah 3 — Kenali Struktur Folder

Setelah proyek berhasil dibuat, kamu akan melihat struktur folder seperti ini:

📁 nama-proyek-kamu/
├── 📁 app/
│ ├── 🌐 favicon.ico
│ ├── 🎨 globals.css ← CSS global
│ ├── ⚡ layout.js ← ✅ .js bukan .tsx!
│ ├── ⚡ page.js ← halaman utama
│ └── 🎨 page.module.css
├── 📁 public/
├── 📄 .eslintrc.json
├── 📄 jsconfig.json ← ✅ bukan tsconfig!
├── ⚡ next.config.mjs
└── 📄 package.json
💡 Perhatikan! Semua file menggunakan extension .js (JavaScript), bukan .tsx (TypeScript). File konfigurasi juga jsconfig.json bukan tsconfig.json. Ini artinya kamu sudah benar memilih JavaScript saat instalasi!

▶️ Langkah 4 — Jalankan Development Server

Masuk ke folder proyek dan jalankan development server:

Terminal — jalankan proyek
cd nama-proyek-kamu && npm run dev

Kalau muncul halaman seperti di atas, SELAMAT! 🎉 Setup berhasil!

✏️ Langkah 5 — Edit Halaman Pertama

Buka file app/page.js di VS Code, hapus semua isinya, lalu ganti dengan kode sederhana ini:

app/page.js — Halaman pertama kamu!
export default function Home() {
  return (
    <main>
      <h1>🎓 Halo Mahasiswa!</h1>
      <p>Ini proyek Next.js pertama saya.</p>
      <p>Menggunakan <strong>JavaScript</strong>, bukan TypeScript.</p>
    </main>
  );
}

Simpan file → browser auto-refresh → teks baru muncul! 🎉

💡 Hot Reload: Next.js punya fitur Fast Refresh — setiap kali kamu menyimpan file, browser otomatis memperbarui halaman tanpa perlu refresh manual. Keren, kan?

🛠️ Perintah-Perintah Penting

Berikut perintah-perintah yang sering kamu gunakan saat bekerja dengan Next.js:

npm run dev

Jalankan dev server (untuk ngoding)

npm run build

Build untuk production

npm start

Jalankan hasil build

npm run lint

Cek kualitas kode

⚠️ Troubleshooting

Masalah umum yang sering muncul dan cara mengatasinya:

"node: command not found"
Restart terminal setelah install Node.js. Di Windows, pastikan Node.js sudah ditambahkan ke PATH saat instalasi.
"Port 3000 already in use"
Kill proses lain yang menggunakan port 3000, atau jalankan di port berbeda: npm run dev -- -p 3001
File berekstensi .tsx bukan .js
Kamu salah pilih TypeScript saat create-next-app. Hapus folder proyek, lalu ulangi perintah dan pilih No untuk TypeScript.
"EACCES permission denied"
Windows: Run terminal as Administrator.
Mac/Linux: Gunakan sudo di depan perintah, atau perbaiki permission npm.

✅ Checklist Verifikasi

Centang semua item di bawah ini untuk memastikan setup kamu sudah benar. Progress tersimpan otomatis!

  • ✓ Node.js terinstall (versi 18+)
  • ✓ npm terinstall
  • ✓ Proyek Next.js berhasil dibuat (JavaScript, bukan TypeScript)
  • ✓ npm run dev berhasil jalan
  • ✓ Halaman muncul di localhost:3000
  • ✓ Berhasil edit app/page.js dan melihat perubahan
